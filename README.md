# E-Commerce Retention Analysis Dashboard

## Project Overview
This Power BI dashboard project analyzes e-commerce transaction data to identify revenue trends, customer behavior, product performance, returns, and cancellations.

The dashboard was built using Excel, Power Query, and Power BI to transform raw transactional data into interactive business insights.

---

## Key Insights
- Monthly revenue trend analysis
- Top revenue-generating products
- Customer distribution by country
- Revenue impact of product returns
- Revenue impact of order cancellations

---

## Tools Used
- Power BI
- Power Query
- Excel
- Data Cleaning & Transformation
- Data Visualization
